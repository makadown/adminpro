Este proyecto es un demo con funcionalidad CRUD de usuarios, medicos y hospitales. También proporciona edición del perfil del usuario conectado.
  • Tiene 2 roles: ROLE_ADMIN y ROLE_USER
  • Unicamente el rol de admin puede editar usuarios
  • Tiene autenticación normal y por google
  • Front End : Angular 5 (Github pages)
  • Back End : Express, Node JS, MongoDB (Heroku)
  • Por la forma en que se programó la subida de archivos, éstos tendrán un corto periodo de existencia en server (Limitación de Heroku).
  • Pendiente funcionalidad de consultas, pacientes y su seguimientos.
